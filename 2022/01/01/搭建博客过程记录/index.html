github准备

  1. 首先要注册一个github账号。注册账号的时候会有一个用户名(UserName)，这个用户名不是你的邮箱。

  2. 创建一个仓库。仓库的名称是 用户名+github.io

本地生成ssh密钥

  1. 执行 ssh-keygen -t rsa -C "邮箱地址"，这个邮箱地址是上面注册github时填写的邮箱地址。
  2. 密钥的默认路径是：~/.ssh (win请自行百度)。
  3. 使用任意编辑器打开。~/.ssh/id_rsa.pud文件。看到的字符串就是刚刚生成的密钥。

把生成的SSH密钥配置到github

  1. 登录github
  2. 找到 Settings -> SSH adn GPD keys
  3. 单击 New SSH key 。
  4. 在Title中起个名字。
  5. Key中复制~/.ssh/id_rsa.pud文件中的字符串。点击 Add SSH key
  6. 完成之后输入 ssh -T git@github.com 测试是否配置成功。

    看到
    Hi XXXX! You’ve successfully authenticated, but GitHub does not # provide shell access.
    则表示配置成功。
