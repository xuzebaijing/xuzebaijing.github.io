Hexo+Next+github搭建自己的小博客

发表于 2019-10-26

记录一下自己搭建博客的过程

上班工(hua)作(shui)的时候看到几个漂亮的博客，感觉作为朴实的农民，自己也得有一个博客。于是在强大的编(bai)程(du)天(jiao)赋(cheng)下,开始搭建自己的博客。

1.准备工作

  • 注册一个github账号，然后创建一个仓库,仓库的名称为 ： github的用户名+.github.io(这里用户名不是github的昵称)
  • 本地安装 node
  • 本地安装 git

    1.1 生成SSHkey

  • 使用以下命令
  • 然后连续三次回车，最终会生成一个文件在用户目录下，打开用户目录，找到 .ssh/id_rsa.pub 文件用 记事本打开此文件。
  • 打开github个人主义，进入个人设置 -> SSH and GPG keys -> New SSH key
  • 测试是否成功
  • 提示Are you sure you want to continue connecting (yes/no)?，输入yes，然后会看到：

    Hi liuxianan! You’ve successfully authenticated, but GitHub does not provide shell access.

  • 说明成功

2.开始安装Hexo

  • 在本地创建一个文件夹这里我创建的文件夹名为blog，后面将会用blog来演示。
  • 先安装hexo,使用官方推荐使用npm命令安装，如果嫌弃npm速度慢的话，可以使用cnpm
  • 进入blog在空白处单击右键打开git Bash here(推荐使用)
  • 输入 hexo init 开始创建 Hexo项目
  • 配置项可以参考 Hexo官方文档 传送门

    3.安装主题

  • 这里以Next为例子(方法都是一样的只有主题配置不同)
  • 下载netx: git clone https://github.com/iissnan/hexo-theme-next themes/next
  • 把下载的文件放入 Hexo 创建的 themes 文件夹中
  • 打开Hexo 配置文件 _config.yml 此文件在 blog 目录下
  • 找到 theme 项
  • 配置项可以参考 Next官方文档 传送门
  • 注意一下主题的配置文件和Hexo的配置文件名称相同

    部署
